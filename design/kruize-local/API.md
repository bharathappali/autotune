### IMPORT DATA 

The Import data API takes in a data source URL and type (promethues supported currently) and returns a JSON which is generated by reading the unique cluster group, clusters, namespace, workloads and containers details from data source.

##### Request

```
curl http://<KRUIZE_URL>:<PORT>/monitoring/import_data?datasource_url=<URL>&datasource_port=<PORT>&type=prometheus
```
